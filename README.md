# Bank Account System 🏦

CLI-based banking app in Python — create accounts, deposit, withdraw, and earn interest on savings.

# Features
- Current and savings accounts
- Savings accounts auto-apply interest on creation
- Deposit / withdraw with balance and amount validation
- Account listing with formatted balances
- Handles invalid input cleanly — no crashes

# Concepts used
Classes & objects · Inheritance (super()) · Encapsulation (private __balance)
__str__ · Dictionaries · Input validation · DRY helper functions

# How to run
```
python bank.py
```

# Menu
1. Create current account
2. Create savings account
3. View account (deposit/withdraw)
4. List all accounts
5. Exit

# What I learned
Refactoring working code to remove duplication, separating input
handling from business logic, and using inheritance properly between
related classes.
